Basic Example - SocketIO
========================

Description:
-------------
Basic example

Requirements:
-------------
1. Python 3.x
2. flask and flask-socketio
  `pip install -r requirements.txt`

Getting Started
---------------
1. clone this repository:
  `git clone git@github.com:jack-factor/socket_example.git`
2. start app
  `FLASK_APP=run.py flask run`
3. go to
  `http://127.0.0.1:5000/`
